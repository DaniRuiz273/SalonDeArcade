Proyecto Salón De Arcade

El programa simula de manera muy sencilla el funcionamiento de un salón arcade. Al iniciar nuestra máquina nos saldrá un menu donde podemos interactuar con él. 
En este menu podemos realizar varias acciones, desde registrar un nuevo jugador en la sala o registrar una máquina nueva en la sala, hasta poder jugar a cualquier 
máquina que este en la sala. También podrás ver el ranking de las mejores puntuaciones de cada máquina como la personas que ha realizado dicha puntuación.
